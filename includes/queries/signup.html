Gestion des inscriptions

Description rapide: Cette page permet d'inscrire un utilisateur et de l'enregistrer dans la base de données. Elle se divise en 5 sections, définies par la variable $_POST['method']:

  • Inscription par Facebook
  • Inscription par Github
  • Inscription par Google
  • Inscription par Twitter
  • Inscription par Let's Dev !

Type de données renvoyées: Tableau JSON de la structure suivante:

  • status (string): Statut de la requête (success: la requête s'est bien passée et l'utilisateur a été ajouté à la base de données; error: il y a eu une erreur lors de la requête)
  • messages (array): Liste des messages d'erreur ou de succès

Attention: N'oubliez pas de renseigner $_POST['method'], sinon, une erreur sera générée et le tableau renvoyé sera illisible.

Cette section permet d'ajouter un utilisateur à la base de données sans passer par un service externe.

Le script vérifie que les données facultatives sont saisies, sinon il leur donne une valeur par défaut.

Le script vérifie ensuite que toutes les données obligatoires sont saisies.

Ensuite, le mot de passe et sa confirmation sont comparés pour voir s'ils sont les mêmes.

Enfin, l'adresse e-mail, le nom et le prénom sont cherchés dans la base de données pour voir si l'utilisateur existe déjà.

Dans le cas où tout est correct, une requête à la base de données est effectuée pour inscrire l'utilisateur.
